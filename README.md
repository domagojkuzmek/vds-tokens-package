
# @verda-systems/web-tokens

## Installation

Install my-project with npm

```bash
  npm i @verda-systems/web-tokens --save-dev
```
    
Obtain the tokens from the repository:

```bash
  import tokens from "@verda-systems/web-tokens/build/tokens.js"
```

Apply the tokens, for example

```bash
tokens.color.tourmaline[100]
```
